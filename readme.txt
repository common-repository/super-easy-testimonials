==Description==
Super Easy Testimonials adds flexibility to your wordpress site in creating and managing testimonials. 
[You can edit the testimonial your own way if you are a wordpress developer]. Cheers!

==Short code==
Yes we have got a magic shortcode for you. Just create a page add the short code, you are done.
The testimonials get displayed in your website front end.
Use short code [list-testimonials limit="5"] to display the testimonials.

==Installation==

Upload the zip file, activate it, and boom!!!! You are ready to write your first testimonial.
